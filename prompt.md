这是一个监控某个 QQ 群的网页监控，你需要完成：

- 提供给机器人端的上报 API
- 提供给用户的数据面板

# 群数据网页监控系统

## 功能

你需要展示以下数据

- 群成员数量（包括目前成员数，每日新增成员数，每日离开成员数，最新进群的 5 个人及时间）
- 成员发言统计（按时间段统计）

目前就这点功能，后续可能会增加

## API 设计

### 上报 API

- 接口地址：/api/report
- 请求方法：POST

请你设计剩下内容

## 技术栈

- 后端：Go + Gin
- 前端：Vue 3 + Vite + TailWind CSS
- 数据库：PGSQL（当前开发阶段使用 SQLite，但是要做好预留）

大胆的渐变文字、动态光晕背景、浮动装饰元素和圆形边框装饰
无限滚动的数据统计展示，营造动感氛围
现代化的不对称网格布局，展示各项专业能力，带有交互悬停效果
大字号数字引导的流程展示，清晰的层次结构

整体采用黑色为主、白色文字、蓝色点缀的配色方案，所有组件都具有流畅的悬停动画和响应式设计。

开发过程中开发文档输出到 .agent/docs/当前任务名/步骤名.md 文件中
所有的调试测试脚本输出到 .agent/scripts/当前任务名/步骤名/ 文件夹中

如果你需要执行任何命令，你都需要输出到 .agent/do.bat 文件中，并调用相同的命令执行这个文件
