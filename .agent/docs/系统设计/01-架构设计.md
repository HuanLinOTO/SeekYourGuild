# 群数据网页监控系统 - 架构设计

## 项目概述

SeekYourGuild 是一个 QQ 群数据监控系统，用于收集、存储和展示群成员动态数据。

## 系统架构

```
┌─────────────────┐     ┌─────────────────┐     ┌─────────────────┐
│   QQ 机器人端   │────▶│   后端 API      │────▶│   数据库        │
│   (数据上报)    │     │   (Go + Gin)    │     │   (SQLite/PGSQL)│
└─────────────────┘     └─────────────────┘     └─────────────────┘
                               │
                               ▼
                        ┌─────────────────┐
                        │   前端面板      │
                        │   (Vue 3)       │
                        └─────────────────┘
```

## 技术选型

| 层级        | 技术                | 说明                        |
| ----------- | ------------------- | --------------------------- |
| 后端框架    | Go + Gin            | 高性能 HTTP 框架            |
| 数据库      | SQLite / PostgreSQL | 开发用 SQLite，生产用 PGSQL |
| ORM         | GORM                | Go ORM 框架，支持多数据库   |
| 前端框架    | Vue 3 + Vite        | 现代化前端构建              |
| CSS 框架    | Tailwind CSS        | 原子化 CSS                  |
| 状态管理    | Pinia               | Vue 3 官方状态管理          |
| HTTP 客户端 | Axios               | 前端 API 调用               |

## 数据流

1. **数据上报流程**

   - QQ 机器人监听群事件
   - 通过 POST /api/report 上报数据
   - 后端验证并存储到数据库

2. **数据展示流程**
   - 前端定时请求数据接口
   - 后端聚合查询数据库
   - 返回格式化的统计数据
   - 前端渲染动态面板

## 安全设计

- API Key 验证上报请求
- CORS 配置限制跨域
- 请求频率限制
